# Visión General de Coderic.org

## La Incubadora

Coderic.org es una incubadora completa de proyectos de software. El flujo es:

1. **Development** → Idea a Plan estructurado
2. **Crowdfunding** → Plan a Financiamiento
3. **Coworking** → Financiamiento a Espacio de trabajo
4. **Freelancers** → Espacio a Equipo de trabajo
5. **Learning** → Equipo a Conocimiento y autonomía
6. **Community** → Todo lo anterior conectado

## El Modelo de Negocio

- **NO vendemos** servicios, coworking, cursos, comisiones de freelancing
- **SÍ facilitamos** que desarrolladores creen software
- **El software se comercializa** a través de:
  - Coderic Cloud (infraestructura)
  - Coderic Hub (integración de contenidos)
  - Coderic Fintech (pagos y finanzas)
  - Coderic Business (soluciones empresariales)
- **Revenue sharing** del éxito comercial con los desarrolladores

## Principio Fundamental

"Nuestro negocio es la confianza y el recurso humano"

La palabra de Neftalí Yagua es su ley:
- No hay lipsum ni contenido falso
- No gritamos con mayúsculas
- Si algo es importante, habla por sí mismo
